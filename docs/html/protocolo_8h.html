TPMENSAGEM analisarComando ( char *  mensagem,
int  is_serv 
)

Verifica integridade da mensagem e retorna o comando/dados relevantes.

Retorna o comando que deve ser eviando pela ponta da comunição que recebeu o comando. Exemplos: CASO 1: Cliente -> OpenValve#123#999! Resposta: retorno : TPMENSAGEM = int valor = 999; (Enviar(&sock,&server,&from,length,fromlen,buffer)<0) { int sequencia =123;
int comando = 16; -> servidor deve abrir a válvula CASO 2: Servidor -> Open#123! Resposta: retorno : TPMENSAGEM = int valor = -1; int sequencia =123;
int comando = 6; -> cliente recebeu um ACK

Parameters
mensagemEndereço que contem a messagem (char array)
is_servDefine em qual ponta da comunição a função é chamada
Returns
TPMENSAGEM que foi detectado com base na mensagem recebida.
